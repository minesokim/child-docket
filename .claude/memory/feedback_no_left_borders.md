---
name: No Left Border Accents or Vibe-Coded Icons
description: David hates left-border accent lines on cards, generic icons, and any default AI pattern. Use actual brand logos.
type: feedback
---

Specific patterns David rejected in session 4:
- Left-border accent lines on calendar event boxes ("always defaulting to this stupid little design with the line on the left hand side")
- Calendar icon on upcoming call banner ("vibe coded, gross")
- Generic Video icon instead of actual Google Meet / Zoom logos
- "AI-DRAFTED MESSAGE FROM ANTONIO" label with bot icon on compose areas

**Why:** These are the most common "AI-generated dashboard" clichés. They immediately signal the design wasn't intentional.

**How to apply:** Use clean rounded boxes with soft fill colors. Use actual brand logos (Google Meet, Zoom) not Lucide icons. Compose areas should match the app's standard input style — no special "AI" framing. Money should use `font-bold tabular-nums` sans-serif, never serif `font-display` for numbers.
