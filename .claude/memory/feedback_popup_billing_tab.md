---
name: Popup Billing Tab - No Redundancy
description: Don't add BillingCard to popup overview since the popup already has a dedicated billing tab
type: feedback
---

The client detail popup has 6 tabs: overview, intake, documents, messages, billing, notes. The full page has billing integrated into the overview (no tabs).

**Why:** Adding billing to the popup overview would duplicate the billing tab content.

**How to apply:** In the popup, billing stays in the billing tab only. In the full page (which has no billing tab), billing is shown on the overview using the shared BillingCard component.
