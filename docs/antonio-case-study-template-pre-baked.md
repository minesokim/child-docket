   - Apply the anonymization checklist from `docs/antonio-case-study-interview.md` §2 Step 1:
     - Client name → "Client A" / "Client B"

     - Geographic location → County or region only
     - AGI → bucket only
     - Adjustment dollars → bucket only
     - Dates → month + year only
