## Position statement

A paid preparer who claims the Child Tax Credit (CTC), Additional Child Tax Credit (ACTC), or Credit for Other Dependents (ODC) MUST complete Form 8867 due-diligence checklist for these credits AND retain documentation. Failure triggers a separate $650 penalty PER FAILURE (additive to any EITC due-diligence penalty on the same return).

## Plain-English description

CTC due-diligence is **identical in structure** to EITC due-diligence (p008) but for a different credit. The same Form 8867 covers EITC + CTC + AOTC + HOH; each missed checklist is a separate $650 penalty.

**Credit overview (2025 + TCJA expiration impact)**:
- **CTC (Child Tax Credit)**: $2,000 per qualifying child under age 17. Phases out above $400,000 MFJ / $200,000 other. Up to $1,700 refundable as Additional Child Tax Credit (ACTC) for 2025. **Sunsets to $1,000/child after 12/31/2025 unless Congress extends TCJA**.
- **ODC (Credit for Other Dependents)**: $500 per non-CTC-qualifying dependent (older children, qualifying relatives). Same phaseouts as CTC.
- **ACTC refundable portion**: requires earned income (not pure investment-income households).

**Why a separate due-diligence entry from EITC**: the §32(c) eligibility tests (EITC qualifying child) differ from §24 eligibility (CTC qualifying child). A child may qualify for EITC but not CTC (e.g., 17-year-old qualifies for EITC, not CTC due to age cap), or vice versa.

**Why an EA cares**: nearly every return with children claims CTC. The penalty math compounds: an MFJ with 3 children claiming EITC + CTC + (potentially AOTC if college-age) on one return is **3-4 separate §6695(g) checklists**. Miss all 3-4 = $1,950-$2,600 in penalties from ONE return.

## Fact pattern triggers

The Discovery agent should consider this position when:
- Form 1040 Line 19 (CTC + ACTC) > 0
- Form 1040 Line 20 + ODC > 0
- Form 8867 attached
- Dependents listed on Form 1040 (qualifying children + qualifying relatives)
- AGI within or below CTC phaseout range
- Earned income on return (gates the ACTC refundable portion)
- Children near the age 17 cutoff (CTC eligibility shifts to ODC)
- Dependents who are qualifying relatives (non-qualifying-child ODC analysis)
- Multi-generational households

## The four §6695(g) due-diligence requirements (Treas. Reg. §1.6695-2(b)) applied to CTC + ODC

### Requirement 1: Knowledge / reasonable inquiry

Trigger inquiries when:
- Multiple non-spouse adults in the household claim the child
- Child has split residency (custody arrangement; check §152(e) tiebreaker rules)
- Dependent is a non-traditional household member (e.g., girlfriend's child without legal-parent step)
- Child's age suggests CTC vs ODC distinction (e.g., born December near the 17-year-old cutoff)
- Qualifying relative income approaches gross-income threshold ($5,050 for 2024; updated annually)
- Multi-generational household with grandparent + parent + child (potential tiebreaker)

### Requirement 2: Computation

- Apply correct CTC vs ODC distinction (under 17 = CTC; 17+ = ODC)
- Apply AGI phaseout ($400K MFJ / $200K other) correctly
- Compute ACTC refundable portion based on earned income (15% of earned income above $2,500, capped at $1,700/child for 2025)

### Requirement 3: Form 8867 completion

- Complete Form 8867 Part II (CTC/ACTC/ODC) for each child claimed
- Attach to the return + file

### Requirement 4: Documentation retention

3 years from later of due date / filing date. Items:
- Form 8867
- Computation worksheets
- Documents supporting relationship + residency + age of each child
- Records of inquiries made (residency tiebreaker analysis if applicable)

## Required substantiation (the audit-defense file)

1. **Completed Form 8867** (Part II for CTC/ODC)
2. **Computation worksheet** (CTC/ACTC formula application)
3. **Birth certificate** or other age verification for each qualifying child
4. **Residency documentation** (school records, medical records, custody documents)
5. **Relationship verification** (birth certificate, adoption decree, etc.)
6. **Custody arrangement documentation** if applicable (tiebreaker analysis)
7. **Phaseout analysis** if AGI in range
8. **3-year retention log**

## Common audit-defense framing

When IRS audits CTC due-diligence (preparer audit):

1. **Produce Form 8867 Part II** for each return at issue.
2. **Produce age + residency + relationship documentation** for each qualifying child.
3. **Show inquiry documentation** for any return where information seemed inconsistent.
4. **Show computation worksheet** with CTC vs ODC distinction correctly applied.
5. **Show 3-year retention** intact.

Typical IRS preparer-audit questions:
- "Show me Form 8867 Part II for [client]'s [year] return."
- "Show me birth certificate for child [Name]."
- "Show me residency records (school, medical) for child [Name]."
- "Did you make any inquiries about custody arrangement?"
- "Why was CTC applied vs ODC? Show me the age verification."

Docket pre-logs each.

## Common mis-uses / failure modes

| Mis-use | Why it fails | Penalty |
|---|---|---|
| Form 8867 Part II not completed | §1.6695-2(b)(3) fails | $650/failure |
| Claiming CTC for 17-year-old (ODC applies) | §24(h)(4) explicitly distinguishes; age tied to year-end status | $650/failure + §6694(a) |
| Claiming CTC for child who lived elsewhere >half year | §152(c)(1)(B) residency test fails | $650/failure + §6694(a) |
| Multiple claimants on same child without §152(e) tiebreaker | Allocation rule fails | $650/failure |
| Claiming ODC for qualifying relative whose gross income > $5,050 (2024) | §152(d)(1)(B) gross-income test fails | $650/failure + §6694(a) |
| Claiming both CTC and ODC for same person | Mutually exclusive | $650/failure + §6694(a) |
| Improperly computing ACTC refundable portion | Earned-income formula error | $650/failure |
| Phaseout not applied above AGI threshold | §24(h)(2) reduction not applied | $650/failure + §6694(a) |
| Documentation not retained for 3 years | §1.6695-2(c) retention fails | $650/failure |
| Knowingly claiming CTC for ineligible child | Substantive fraud | $650/failure + §6694(b) + §7206 |
