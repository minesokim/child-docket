## Position statement

A paid preparer who claims the Earned Income Tax Credit (EITC) on a client's return MUST complete Form 8867 (Paid Preparer's Due Diligence Checklist) and retain documentation supporting the credit's eligibility, computation, and amount. Failure to meet ANY of the four due-diligence requirements triggers a $650 penalty PER FAILURE.

## Plain-English description

§6695(g) is the load-bearing preparer penalty in modern tax practice. The 2026 penalty is **$650 per failure**, per Rev. Proc. 2025-32 (inflation-indexed annually). A failure can be ANY of:

1. **Failure to complete Form 8867** (or doing it incorrectly / incompletely)
2. **Failure to compute the credit** based on accurate information
3. **Failure to make reasonable inquiries** when information seems inconsistent
4. **Failure to retain documentation** for 3 years (longer if litigation pending)

A single return with EITC + CTC + AOTC + HOH = **4 separate checklists**. Missing all 4 on one return = $2,600 in penalties FROM ONE RETURN.

**The IRS started auditing §6695(g) compliance aggressively in 2024.** Audit selection focuses on:
- Preparers with high EITC return volume
- Returns where credit appears questionable on its face
- Preparers with prior §6695(g) findings
- Returns where EITC is claimed with self-employment income (high-error-rate category)

**Why this is Tier 1 procedural**: the checklist requirement is mandatory and binary. Either the checklist was completed properly, or it wasn't. There's no judgment call. Confidence in the position = confidence in the documentation pack.

**Why an EA cares**: this is the single highest-frequency preparer penalty exposure. Antonio's segment (2-10 preparer firms with Latino-business or low-income client mix) hits EITC due-diligence on 30-60% of returns. A pattern of §6695(g) findings can trigger PTIN suspension.

## Fact pattern triggers

The Discovery agent should consider this position when:

## 4-tier confidence rationale

**Tier 1 (Settled — procedural)** for the completed-checklist position: Form 8867 fully completed with each of the 4 requirements met, documentation retained.

The §6695(g) framework is binary. The checklist either gets done correctly, or it doesn't. There's no Substantial Authority middle-ground for a partial checklist.

**Where the framework downgrades**:

- **REFUSED below RB**: claiming EITC when the client doesn't qualify (e.g., relationship doesn't meet §32(c) standard; AGI exceeds threshold; SSN requirements not met). Refuse the credit.
- **Pre-emptive refusal**: client requests EITC for a clearly-non-qualifying child (e.g., niece who didn't live with taxpayer >half the year). Refuse on the front end.

## The four §6695(g) due-diligence requirements (Treas. Reg. §1.6695-2(b))

Every Form 8867 must capture:

### Requirement 1: Knowledge requirement

The preparer must:
- Apply common-sense knowledge + ordinary business understanding
- Make reasonable inquiries when information seems incorrect, inconsistent, or incomplete
- Document the inquiries + responses

Examples of inquiries triggered:
- Filing status = HoH but address doesn't match qualifying child's
- Schedule C income reported but no business records
- Multiple qualifying children with potentially different relationships
- Self-employment with no expenses (suspicious profit pattern)
- Income too low to maintain documented living arrangements

### Requirement 2: Computation requirement

The preparer must:
- Compute the credit based on the information provided
- Use IRS-published EITC tables / formulas correctly
- Apply the §32 phaseout based on AGI

### Requirement 3: Form 8867 completion

The preparer must:
- Complete Form 8867 fully (every checkbox + every question)
- Attach Form 8867 to the return
- File Form 8867 with the return

### Requirement 4: Documentation retention

The preparer must retain for **3 years** after the later of:
- The due date of the return
- The date the return was filed
- The date the preparer first knew of the credit claim

Retained items must include:
- Copy of Form 8867
- Worksheets used to compute the credit
- Copies of any documents provided by the taxpayer that the preparer relied on
- Record of inquiries made + responses + relevant facts

## Required substantiation (the audit-defense file)

1. **Completed Form 8867** for the tax year
2. **EITC computation worksheet** (Schedule EIC if applicable)
3. **Documentation of inquiries made** when information seemed inconsistent (text messages, email, notes in client file)
4. **Documentation of taxpayer-provided records** (birth certificates, school records, custody documents, residency records)
5. **Filing-status verification** if HoH (Form 8867 Part IV)
6. **3-year retention log** documenting where the §6695(g) records are stored
7. **Year-over-year consistency check** (did the EITC eligibility change? if so, why?)

## Common audit-defense framing

When IRS audits due-diligence compliance (this is a **preparer audit**, not just a taxpayer audit — the IRS comes to the preparer directly):

1. **Produce the Form 8867** for each EITC return at issue.
2. **Produce the documentation pack** (worksheets, inquiry records, taxpayer-provided documents).
3. **Show the inquiry chain** for any return where information seemed inconsistent.
4. **Show the 3-year retention** is intact.
5. **Show year-over-year consistency** (did anything change in eligibility?).

Typical IRS preparer-audit questions:
- "Show me Form 8867 for [client]'s [year] return."
- "Show me the records supporting the qualifying child claim."
- "What inquiries did you make? Where are they documented?"
- "How long do you retain these records?"
- "Why did [client] qualify in [year-1] but not [year-2]?" (or vice versa)

## Common mis-uses / failure modes

| Mis-use | Why it fails | Penalty |
|---|---|---|
| Form 8867 not completed | Treas. Reg. §1.6695-2(b)(3) fails | $650/failure |
| Form 8867 completed but checkboxes not actually corresponded to facts | Substantive fail; reasonable-inquiry requirement breached | $650/failure |
| Inquiries not documented when information seemed inconsistent | §1.6695-2(b)(3) knowledge requirement fails | $650/failure |
| Documentation not retained for 3 years | §1.6695-2(c) retention fails | $650/failure per return audited |
| Claiming EITC for a child who doesn't meet §32(c) qualifying-child standard | Substantive claim error | $650/failure + §6694(a) |
| Filing-status documentation incomplete (HoH) | §32(c) eligibility fails | $650/failure (separate HoH checklist) + §6694(a) |
| Claiming EITC with prior §32(k) ban in effect | §32(k) restriction | $650/failure + §6694(b) potentially |
| Self-employment Schedule C with no business records but EITC claimed | Pattern indicates fabricated income; §6695(g) + §6694(b) + §7206 risk | Maximum penalties |
| Inflating income to qualify for EITC (working the phase-in curve) | §6694(b) reckless | $5,000+ |
| Suppressing income to qualify for EITC | Same | $5,000+ |
